An attestion macro that checks a boolean condition cond.
On failure, the attestion prints an error message and exits the program with a non-zero status code. The message always includes the source code location of the attestion and a stringified version of the condition that failed.

The first argument is the condition to check, the (optional) rest are passed to std::format to form a custom extra message that is printed if the condition fails.
